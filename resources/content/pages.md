Create a page
-------------

{% highlight php %}
POST 	https://api.create.net/pages
{% endhighlight %}

### Input

* *page_title* string
* *page_type* standard, dynamic, homepage
* *content* string
* *titletag* string
* *meta_keywords* string
* *metadesc* string
* *isonmenu* boolean
* *menu_text* string
* *page_filename* string
* *menu_order* int
* *parent_id* int

### Response

{% highlight php %}
Status: 201 Created
Location: http://api.create.net/pages/236
{% endhighlight %}
